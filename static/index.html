About Me


Hi, you can call me Josh.

My Projects


  • Ubuntu Cinnamon

    • Ubuntu Cinnamon

    Ubuntu Cinnamon is an official flavor of Ubuntu Desktop. Instead of shipping with GNOME Shell, it comes with the Cinnamon desktop environment and its components.


    I had the idea around February 2019. The first release was in Deceember 2019 (19.10 "Eoan Ermine"), and Ubuntu Cinnamon became a flavor in time for the April 2023 ("Lunar Lobster") release.

  • Ristretto

    • Ristretto

    Ristretto is a plugin for Aroma (Wii U custom firmware) that allows for the Nintendo Wii U to join the Internet of Things. It is the integration of smart home technology with the Wii U.


    Ristretto runs a HTTP server on the console that responds to appropriate requests.


    Started in August 2024, just about a month later I was able to control the Wii U via HomeKit (except power on - it is still a WIP).
