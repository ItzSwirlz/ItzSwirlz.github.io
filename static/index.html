  • Ristretto

    • Ristretto

    Ristretto is a project that makes the Nintendo Wii U join the Internet of Things.


    With custom firmware (Aroma, for the Wii U), the Ristretto plugin runs a HTTP server on the console that responds to appropriate requests.


    Started in August 2024, just about a month later I was able to control the Wii U via HomeKit (except power on - it is still a WIP).
